Bugsnag JS: Typescript example with inline sourcemaps

This page is a basic example of how to include bugsnag-js on a Typescript page via a <script> tag.

The "app.ts" is compiled with tsc --inlineSourceMap --inlineSources and served with a link to the sourcemap so that Bugsnag can fetch the sources/maps when an error is received.
